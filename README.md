🎓 University Management System

A desktop-based University Management System developed using Core Java (Swing & AWT) to efficiently manage academic and administrative tasks. The application provides a user-friendly interface for handling student details, faculty records, examination data, and more.

🚀 Features:
->👨‍🎓 Add / View Student Information
->👨‍🏫 Add / View Faculty Details
->📝 Examination & Marks Management
->📅 Leave Management System
->💰 Fee Management
->🔄 Update Records (Student & Faculty)
->🧾 Detailed Record Viewing
->🛠 Utility Tools (Calculator, Notepad integration)

🛠 Tech Stack:
->Language: Java
->GUI: Swing, AWT
->Database: MySQL
->Connectivity: JDBC
->IDE: Eclipse

🧠 Key Concepts Used:
->Event Handling (ActionListener)
->Object-Oriented Programming (OOP)
->JDBC Connectivity
->GUI Design with Swing & AWT
->Exception Handling

⚙️ Setup Instructions
1.Clone the repository
2.Import project into Eclipse
3.Add required JAR files:
4.MySQL Connector
5.JCalendar (for date picker)
6.Configure MySQL database
7.Run Project.java
